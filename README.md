# Haroon Khalid - Portfolio

Welcome to my personal portfolio website! This repository contains the source code for my portfolio, showcasing my skills, experience, and projects in Software Engineering, AI, and Digital Creation.

## Live Demo
[https://haronkhalid.github.io/portfolio/](https://haronkhalid.github.io/portfolio/)

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS with Custom Properties)
- JavaScript (Vanilla JS for DOM manipulation)
- GSAP & ScrollTrigger (For advanced animations and scrolling effects)

## Features
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **Premium Animations**: Smooth scrolling and interactive elements using GSAP.
- **Dynamic Content**: Displays my current projects, technical skills, and professional experience.

## Setup & Local Development
Since this is a static website, no build tools are required. 
1. Clone the repository.
2. Open `index.html` in your favorite web browser.

Alternatively, you can use a local server like VS Code's Live Server extension for a better development experience.
